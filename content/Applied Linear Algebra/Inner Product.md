## Definitions and Properties

The [[Inner Product]] ($\left< x,y \right>$)is defined as a dot product in this subspace.
Some of these properties entail
1. Matrix notation: $\left< x,y \right>=x^Ty$
2. Symmetry: $\left< x,y \right> = \left< y,x \right>$
3. Bi linearity: $\left< x, ay+bz \right> = a\left< a,y \right> + b\left< x,z \right>$
4. Relation to matrix transpose: $\left< Ax,y \right> = \left< x, A^Ty \right>$
5. [[Norm]]: $\left< x,x \right>=\sum_{i=1}^{N}x_{i}x_{i}=||x||^2$
6. [[Cauchy Schwartz Inequality]]: $\left< x,y \right> \leq ||x||||y||$

The most important property in this list is number 4.
>[!Proof]
> Only property 1 is required to show this
> $\left< Ax,y \right> = (Ax)^Ty$ 
> $x^TA^Ty$
> $\left< x,A^Ty \right> = x^T(A^Ty)$
